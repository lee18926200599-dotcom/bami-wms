### 3.2 核心功能模块

#### 3.2.1 危险品混装智能管控模块

**【功能1】危险品类别档案管理**
- 建立9大类危险品基础档案
- 每类维护：特性说明、禁配清单、包装要求、应急措施
- 支持MSDS电子档案上传

**【功能2】禁配规则引擎**
```python
# 混装检查逻辑示例
def check_mixing(dangerous_goods_list):
    """
    检查危险品混装合规性
    """
    forbidden_pairs = {
        '1类': ['所有'],  # 爆炸品严禁混装
        '2类': ['5类', '氧化剂'],
        '3类': ['5类', '氧化剂'],
        '5类': ['有机物', '还原剂'],
        # ... 其他规则
    }
    
    for goods in dangerous_goods_list:
        category = goods.category
        for other in dangerous_goods_list:
            if goods.id != other.id:
                if other.category in forbidden_pairs.get(category, []):
                    return {
                        'result': False,
                        'conflict': f"{category} 与 {other.category} 禁止混装",
                        'goods_a': goods.name,
                        'goods_b': other.name
                    }
    return {'result': True}
```

**【功能3】装车实时冲突检测**
- 装车扫描时自动识别货品危险品类别
- 实时计算与已装车货品的兼容性
- 发现冲突立即弹窗提示并阻止装车
- 生成《危险品装载合规确认单》

**【功能4】运输路径合规校验**
- 根据危险品类别规划合规路线
- 自动避开禁行区域、学校、医院
- 对接交通部门危险品运输备案系统

#### 3.2.2 三级结算与钱包模块

**【价格体系配置】**

```yaml
# 价格体系配置示例
pricing_system:
  routes:
    - route_id: "SH-BJ-001"
      from: "上海"
      to: "北京"

    driver_rate: 0.60    # 司机收入60%
```

**【自动分账逻辑】**

运单签收完成后，系统自动：
1. 计算应收运费 = 基础运费 + 危险品附加费 + 其他费用
2. 扣除平台服务费（15%）→ 进入平台钱包
3. 剩余部分按25%:60%分配给网点和司机
4. 生成三方结算明细，推送至各角色钱包

### 3.3 系统流程设计

#### 3.3.1 标准业务流程

```
【场景】客户下单 → 送货上门

1. 客户下单（密尔克卫平台/直接对接客户ERP）
   - 录入货品信息（名称、重量、体积）
   - 选择危险品类别（1-9类）
   - 上传MSDS文件

2. 系统智能分单
   - 根据目的地匹配最近加盟网点
   - 推送运单至网点后台

3. 网点调度司机
   - 查看司机当前位置与状态
   - 指派司机上门提货
   - 司机APP接收任务

4. 司机提货（关键节点）
   - 扫码核对货品信息
   - 系统自动检查危险品类别
   - 如车辆已有其他危险品，触发混装检查
   - 冲突则阻止装车并提示
   - 无冲突则生成电子运单

5. 运输监控
   - GPS实时定位
   - 电子围栏打卡
   - 异常实时上报

6. 送达签收
   - 电子签名确认
   - 拍照留档
   - 运单状态更新为"已完成"

7. 自动结算（T+0实时）
   - 运费按预设规则拆分
   - 平台钱包 +15%
   - 网点钱包 +25%
   - 司机钱包 +60%
   - 各方收到到账通知
```

#### 3.3.2 分拨中心作业流程

```
【场景】加盟网点 → 分拨中心 → 加盟网点

1. 网点集货
   - 收集本区域待转运货物
   - 系统生成分拨任务单

2. 短驳运输
   - 调派车辆从网点到分拨中心
   - 装车时进行混装检查
   - 到达分拨中心卸货扫码

3. 分拨中心分拣
   - 扫描识别货品目的地
   - 自动分拣至对应区域
   - 生成下一段运输任务

4. 干线/短驳配送
   - 装载前往目的网点货物
   - 再次混装检查
   - 配送至各加盟网点

5. 末端配送
   - 网点接收货物入库
   - 分配给司机进行最后一公里配送
```
