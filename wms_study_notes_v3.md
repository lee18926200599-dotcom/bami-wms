## 一、流程配置 (Workflow Configuration)

### 1.1 业务背景
根据**仓库+货主+配送方式+拣选单类型**配置不同的出库作业流程

### 1.2 配送方式
| 配送方式 | 代码 | 说明 |
|---------|------|------|
| 快递 | KD | 标准快递发货 |
| 快递揽收 | KDLS | 快递员上门揽收 |
| 物流 | WL | 物流配送 |
| 自提 | ZT | 客户自提 |

### 1.3 拣选单类型
| 类型 | 适用场景 |
|------|---------|
| 秒杀单品 | 单件秒杀订单 |
| 秒杀单品多件 | 单SKU多件秒杀 |
| 秒杀多品 | 多SKU秒杀 |
| 单品 | 普通单件订单 |
| 多品 | 普通多SKU订单 |
| 按单拣货 | B2B或大件订单 |

### 1.4 流程节点配置
**标准流程节点**:
1. 单据打印（必选）
2. 拣选单（可选）
3. 拣选单确认（必选）
4. 复核打包（可选）
5. 称重交接（可选）
6. 出库交接（可选）

**特殊显示规则**:
- 快递+单品/多品/按单拣货 → 显示"复核打包"
- 快递+秒杀类型 → 显示"复核打包称重"
- 物流/自提+按单拣货 → 显示"出库交接"

### 1.5 出库回退上游节点
- 可选择从哪个节点回退
- 下拉选项根据已选节点动态刷新

### 1.6 包装材料规格收集
- 需先选择"复核打包"/"复核打包称重"/"称重交接"节点
- 选择收集节点（在哪个节点收集箱型信息）

## 二、打印机配置 (Printer Configuration)

### 2.1 核心字段
| 字段 | 规则 |
|------|------|
| 打印机编码 | 同仓库唯一，20字符 |
| 打印机名称 | 同仓库唯一，50字符 |
| TCP/IP地址 | 同仓库唯一，10字符 |
| 打印文档 | 快递面单/拣选单/交接单 |
| 打印机状态 | 已连接/未连接 |

### 2.2 打印路由管理
- **打印路由名称**: 必填
- **员工**: 有打印权限的员工
- **规则**: 一个员工只能分配一个打印路由

---

## 三、仓库截单时间配置

### 3.1 业务定义
客户与仓库约定：特定时间点前接收的订单必须在规定时间内发出，否则算延迟

### 3.2 配置规则优先级
1. **货主维度** > 仓库维度（货主=全部）
2. 货主未配置或禁用 → 使用仓库维度
3. 均未配置 → 手动控制截单时间

### 3.3 关键字段
- 仓库截单时间: 必填，时间选择器
- 配置编号: YYMMDD+4位流水号（每日从0001开始）

---

## 四、按单拣货判断规则配置

### 4.1 业务目的
系统自动判断哪些订单适合按单拣货（而非波次拣货）

### 4.2 判断条件（至少选一个）
1. **订单品种数 ≥** 正整数
2. **订单总数量（主单位）≥** 正整数

### 4.3 配置优先级
同【仓库截单时间配置】的优先级规则

---

## 五、备货区货品拣货规则配置

### 5.1 业务背景
备货区通常存整箱，拣货区存散件。大批量出库时直接从备货区拣货更高效

### 5.2 核心字段
| 字段 | 规则 |
|------|------|
| 货主 | 必填，当前仓库已授权货主 |
| 商品编码 | 必填，20字符，验证货主归属 |
| 可从备货区拣货：最大单位拣货数量 ≥ | 正整数，最大100,000 |

### 5.3 使用规则
- 货主未配置/禁用 → 该货品不能从备货区拣货
- 已配置且启用 → 按规则执行

---

## 六、库存分配规则配置

### 6.1 业务定义
库存分配可在**接单时预分配**或**波次确认后分配**

### 6.2 分配时机
- 预分配（基于按单拣货判断规则）
- 后分配（波次确认时）

### 6.3 分配顺序结构（后分配时）
- 单品/多品/按单拣货

### 6.4 拣货区库位分配规则
| 规则 | 说明 |
|------|------|
| 清空间优先 | 优先使用空闲库位 |
| 满空间优先 | 优先使用已满库位 |
| 最小仓库空间/序号 | 按空间大小或序号排序 |

### 6.5 备货区分配规则
同上（清空间优先/满空间优先/最小空间）

### 6.6 规则优先级
右侧列表越靠上优先级越高（可调整顺序）

---

## 七、波次单量上限配置

### 7.1 业务目的
防止批量确认时订单量过大导致系统崩溃

### 7.2 核心字段
- **波次单量上限**: 正整数，最大10,000,000
- 未配置或禁用 → 无上限

---

## 八、波次挂起规则配置

### 8.1 业务场景
批次确认时，有订单未成功分配到拣货位库存（缺货订单），选择挂起整个波次或仅挂起缺货订单

### 8.2 挂起类型
- 波次（整波挂起）
- 缺货订单（仅挂起缺货订单，其他生成拣选单）

### 8.3 默认值
未配置或禁用 → 默认"波次"

---

## 九、拣选单生成规则配置

### 9.1 秒杀单品拣选单
| 字段 | 规则 |
|------|------|
| 单量下限 | 正整数 |
| 单量上限 | 正整数，≥下限 |
| 尾单下限 | 正整数，1≤尾单≤下限≤上限 |
| 强制汇总单量下限 | 2≤强制<下限，空=不强制 |
| 每单最大数量 | 正整数 |

### 9.2 秒杀单品多件拣选单
同上字段规则

### 9.3 秒杀多品拣选单
- 单量下限/上限/尾单下限
- 最大品种数
- 每品种最大数量
- 强制汇总单量下限

### 9.4 单品拣选单
- 单量下限/上限/尾单下限
- 最大品种数

### 9.5 多品拣选单
- 先拣后分数量限制: 正整数，最大100

### 9.6 数值校验规则
```
1 ≤ 尾单下限 ≤ 单量下限 ≤ 单量上限 ≤ 9999999
```

## 十八、系统配置使用规则汇总

### 18.1 通用优先级规则
对于所有货主维度配置：
1. 货主维度已配置且启用 → 使用货主配置
2. 货主未配置/禁用，仓库维度启用 → 使用仓库配置
3. 均未配置/禁用 → 按默认规则或手动控制

### 18.2 收货入库规则
| 规则 | 说明 |
|------|------|
| ABC管理 | 实时获取，启用状态，参数=是 |
| 入库货品库存变动点 | 到货登记记录，默认=到货 |
| 仓库分类管理 | 实时获取，启用状态，参数=是 |
| 贵重品管理 | 启用+商品档案启用才管理 |
| 保质期管理 | 规则启用+商品档案启用，收货结果登记必填 |
| 生产批号管理 | 规则启用+商品档案启用 |

### 18.3 库存管理规则
| 规则 | 说明 |
|------|------|
| 允许余料收集 | 出库登记记录参数 |
| 启用出库结果登记 | 当前未使用 |
| 商品辅助信息维护 | 实时获取，启用状态 |
| 超发控制 | 记录出库+超发比例，启用才允许超发 |
| 出库即完成运单 | 仅禁止超发场景适用 |
| 收货结果回传 | 出库登记记录状态=未回传 |

### 18.4 上架管理规则
| 规则 | 说明 |
|------|------|
| 正品是否混批次存储 | 实时获取，默认否 |

### 18.5 销售出库规则
| 规则 | 说明 |
|------|------|
| 仓库-承运商关系 | 订单未指定承运商时预处理后获取 |
| 平台授权 | 直接从集成获取 |
| 承运商编码映射 | 订单转发/预处理/拆包换单时实时获取 |
| 承运商网点对应 | 实时获取启用状态 |
| 面单模板授权门店 | 实时获取 |
| 快递面单模板配置 | 实时获取 |
| 运单号和面单获取 | 实时获取 |
| 流程配置 | 实时获取，修改时漏回调立即回传 |
| 仓库截单时间 | 实时获取 |
| 按单拣货判断规则 | 实时获取 |
| 备货区拣货规则 | 实时获取 |

## 十九、初始化数据

### 19.1 预设流程配置
系统为每个仓库预设默认流程：

| 配送方式 | 拣选单类型 | 预设流程节点 |
|---------|-----------|-------------|
| 快递 | 单品/多品/按单拣货 | 单据打印→拣选单确认→复核打包→称重交接 |
| 快递 | 秒杀类型 | 单据打印→拣选单确认→复核打包称重→称重交接 |
| 揽收 | 按单拣货 | 单据打印→拣选单确认→复核打包 |
| 物流/自提 | 按单拣货 | 单据打印→拣选单确认→复核打包→出库交接 |

### 19.2 其他预设配置
- 下架规则: 默认启用
- 快递面单打印设置: 默认先打/后打组合
- 波次挂起规则: 默认挂起类型=波次

## 二十、配置编号生成规则

**统一规则**: `YYMMDD + 4位流水号`
- 流水号每日从0001开始
- 系统自动生成
- 不可修改

## 关键术语表

| 术语 | 说明 |
|------|------|
| 波次 | 将多个订单合并批量处理 |
| 拣选单 | 指导拣货人员拣货的单据 |
| 秒杀 | 限时抢购活动订单 |
| 先拣后分 | 先批量拣货，再按订单分拣 |
| 复核打包 | 核对商品后打包 |
| 称重交接 | 称重后与快递交接 |
| 尾单 | 波次中剩余数量不足标准批次的订单 |
| 强制汇总 | 强制将多个订单合并成一个波次 |
| RF | Radio Frequency，无线手持设备 |
| 移库 | 库位之间的库存移动 |
| 调拨 | 仓库之间的库存调拨 |
