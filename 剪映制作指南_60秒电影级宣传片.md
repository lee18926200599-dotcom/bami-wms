## 🎬 第三步：新建项目

### 3.1 创建项目
```
打开剪映 → 点击「开始创作」→ 选择「16:9」横版
分辨率：1920×1080 或 3840×2160（4K）
帧率：30fps
```

### 3.2 导入素材
```
点击「导入」→ 选择所有图片+配音文件
将所有素材拖到时间轴上
```

---

## 🎭 第四步：按分镜制作（逐章讲解）

### 【序章】0:00-0:08（8秒）

**画面设置**：
1. 拖入「科技感背景图」到主轨道
2. 点击画面 → 设置时长8秒
3. 添加特效：「动画」→「入场」→「渐显」（2秒）

**文字添加**：
```
点击「文本」→「新建文本」
主标题：八米科技
字体：阿里巴巴普惠体 Bold / 思源黑体 Heavy
字号：120
颜色：白色
动画：「渐显」3秒

副标题：BAMI TECHNOLOGY
字体：DIN Pro / Arial
字号：48
颜色：#0066FF（科技蓝）
动画：「渐显」，延迟1秒出现
```

**特效添加**：
```
点击「特效」→「画面特效」
搜索：「粒子」「光效」「科技感」
推荐：「科技光线」「粒子汇聚」「光斑」
调整透明度：50-70%
```

**配音对齐**：
```
导入配音文件
拖动到音频轨道
点击「识别字幕」自动生成字幕
微调字幕时间轴与画面对齐
```

---

### 【第一章】0:08-0:18（10秒）

**画面设置**：
1. 拖入「八米科技_1_公司概况...」图片
2. 设置时长10秒
3. 添加动画：「组合动画」→「Ken Burns」
   - 起始：100% 居中
   - 结束：110% 轻微右移

**文字添加**：
```
主标题：15年深耕
字号：96，白色，加粗
位置：画面左侧
动画：「从左侧滑入」2秒

数据展示（逐个出现）：
• 1200+ 企业客户
• 100万+ 年订单
• 80% 技术团队

字号：60，颜色：#FF6600（橙色）
动画：「弹入」，每个延迟0.5秒
```

**转场**：
```
在两段之间点击「转场」
选择：「叠化」或「闪黑」
时长：1秒
```

---

### 【第二章】0:18-0:32（14秒）

**画面设置**：
1. 拖入「八米科技_2_AI产品与供应链」
2. 设置时长14秒
3. 分割成两段（各7秒）

**第一段（0:18-0:25）：双齿轮展示**
```
使用「画中画」功能：
• 主轨道：背景图
• 画中画1：齿轮素材（搜索「齿轮动画」贴纸）
• 画中画2：产品列表文字

动画设置：
• 齿轮：「循环旋转」
• 文字：「渐显」逐个出现
```

**第二段（0:25-0:32）：产品矩阵**
```
使用「分屏」或「画中画」：
左侧：供应链产品列表
右侧：AI产品列表

动画：「向中间汇聚」效果
```

**特效**：
```
搜索：「光线」「能量」「科技」
添加「光效扫过」或「能量波动」
```

---

### 【第三章】0:32-0:45（13秒）


## 🎵 第五步：添加音频

### 5.1 导入配音
```
将「chinese_voice_full.mp3」拖到音频轨道
对齐时间轴起点（0秒处）
```

### 5.2 识别字幕
```
点击「文本」→「识别字幕」→「开始识别」
自动根据配音生成字幕

调整字幕样式：
• 字体：思源黑体
• 字号：48-60
• 颜色：白色 + 黑色描边
• 位置：画面下方1/3处
• 背景：半透明黑色（可选）
```

### 5.3 添加BGM
```
点击「音频」→「音乐」→ 搜索
关键词：「史诗」「科技」「大气」「企业"

推荐（剪映内置）：
• 「Victory」风格音乐
• 「Go Time」
• 「Frontier」
• 「Lightning"

调整音量：
• BGM：-18dB（较低）
• 配音：0dB（标准）
```

### 5.4 添加音效
```
搜索音效：「科技」「UI」「转场"
在关键转场处添加：
• 0:08 章节切换：「嗖」声
• 0:32 数据出现：「叮」声
• 0:55 Logo出现：「震撼音效"
```

## 🎨 第六步：调色与特效

### 6.1 整体调色
```
选中所有片段 → 点击「调节」

参数设置：
亮度：-5
对比度：+15
饱和度：+10
色温：-10（偏冷）
色调：+5（偏青）
锐化：+15

暗角：+20（电影感）
```

### 6.2 应用滤镜
```
点击「滤镜」→ 搜索
推荐：
• 「电影」→「青橙」
• 「赛博朋克」（强度50%）
• 「商务」→「深邃"
• 「科幻」→「未来"
```

### 6.3 画面特效
```
在特效轨道添加（贯穿全程）：
• 「电影暗角」
• 「轻微颗粒」（胶片感）
• 「光晕效果」（可选）
```

## 📤 第七步：导出视频

### 7.1 导出设置
```
点击右上角「导出」

格式：MP4
分辨率：1920×1080（或3840×2160）
帧率：30fps
码率：推荐（高画质）
```

### 7.2 多版本导出
```
版本1：完整版 60秒
版本2：30秒精简版（用于朋友圈）
版本3：15秒预告版（用于抖音）
版本4：5秒片头版（用于其他视频开头）
```

### 7.3 文件命名
```
八米科技_品牌宣传片_60s_4K.mp4
八米科技_品牌宣传片_60s_1080P.mp4
八米科技_品牌宣传片_30s_竖版.mp4
```

## 💡 第八步：检查与优化

### 8.1 检查清单
- [ ] 总时长精确60秒
- [ ] 6个章节转场流畅
- [ ] 配音与画面对齐
- [ ] 字幕无错别字
- [ ] 数据准确无误
- [ ] Logo清晰展示
- [ ] BGM音量合适
- [ ] 色调风格统一

### 8.2 预览测试
```
点击「播放」预览全片
在不同设备测试：
• 电脑屏幕
• 手机屏幕
• 投影仪（如需要）
```

### 8.3 修改优化
```
根据预览效果调整：
• 动画速度（太快/太慢）
• 字幕时间（提前/延后）
• 特效强度（太弱/太强）
```

## 🚀 快速制作技巧

### 技巧1：使用模板（最快30分钟）
```
剪映 → 「剪同款」→ 搜索「企业宣传"
选择带「科技感」「大气」标签的模板
替换文字和图片即可
```

### 技巧2：关键帧动画
```
点击画面 → 「关键帧」
设置起始和结束状态
自动生成平滑动画
```

### 技巧3：批量操作
```
选中多个片段 → 右键「批量编辑"
可同时应用滤镜、调色、动画
```

## 📞 遇到问题怎么办？

| 问题 | 解决方案 |
|:---|:---|
| 电脑配置低，卡顿 | 降低预览分辨率，关闭特效预览 |
| 找不到合适素材 | 去「剪映云素材」搜索，或外站下载 |
| 动画效果不满意 | 多尝试不同组合，或简化效果 |
| 配音和画面对不上 | 用「分割」功能逐帧调整 |
| 导出失败/太大 | 降低分辨率或码率，或分段导出 |
