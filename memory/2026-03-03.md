## Key Learnings from Document 9 - Inventory Ledger

### 1. **台账类型体系 (Ledger Types)**

| 台账类型 | 描述 | 匹配维度 |
|---------|------|---------|
| **货位库存台账** | 货主+商品+批次+库位维度的库存 | 9项：仓配商+仓库+货主+库位码+商品码+单位+生产日期+到期日+批次号 |
| **仓库库存台账** | 仓库维度的库存汇总 | 8项：仓配商+仓库+货主+商品码+单位+生产日期+到期日+批次号 |
| **仓库可售库存台账** | 可售库存专用台账 | 同上 |

### 2. **库存变动类型 (Inventory Change Types)**

#### 入库业务：
- **收货确认** (Receipt Confirmation) - 暂存区库存增加
- **上架确认** (Shelf Confirmation) - 从暂存区→货架位
- **收货差异处理** (Receipt Variance) - 报损/报溢

#### 出库业务：
- **库存分配** (Inventory Allocation) - 占用可用库存
- **拣货确认** (Pick Confirmation) - 从拣货位→拣货暂存
- **复核完成** (Review Complete) - 从拣货暂存→复核暂存→发货暂存
- **称重交接** (Weight Handover) - 从发货暂存→出库
- **出库交接** (Outbound Handover) - 最终出库扣减

#### 库内作业：
- **移位** (Transfer) - 从源库位→目标库位
- **移库下架** (Transfer Removal) - 下架到暂存区
- **移库上架** (Transfer Shelving) - 从暂存区上架
- **盘点** (Inventory Check) - 报损/报溢
- **库存差异** (Stock Variance) - 差异处理

### 3. **核心库存概念**

```
物理库存 (Physical Inventory)
  ├─ 可用物理数量 (Available Physical)
  │   └─ 被订单占用 (Order Occupied)
  ├─ 报损占用 (Loss Reported Occupied)
  └─ 移位占用 (Transfer Occupied)

货主可用数量 (Owner Available) = 物理库存 - 占用
```

### 4. **销售出库库存变动流程**

```
1. 订单预处理: 占用仓库可售库存 → 分配货位
2. 拣货确认: 拣货位库存↓ → 拣货暂存库存↑
3. 复核完成: 拣货暂存↓ → 复核暂存↑ → (有后续步骤)发货暂存↑
4. 称重交接/出库交接: 发货暂存↓ → 扣减仓库库存
```

### 5. **拦截/取消时的库存回滚**

| 拦截节点 | 回滚逻辑 |
|---------|---------|
| 等待补货拦截 | 释放可用库存 + 释放货位占用 |
| 汇总待审拦截 | 同上 |
| 拣货确认拦截 | 已拣货→取消暂存；未拣货→释放货位 |
| 复核拦截 | 复核数量→取消暂存；剩余→释放货位 |
| 称重拦截 | 包裹数量→取消暂存 |

### 6. **台账匹配逻辑**

**货位库存台账匹配规则**: 仓配商 + 仓库 + 货主 + 库位码 + 商品码 + 单位 + 生产日期 + 到期日 + 批次号（9项完全匹配）

**仓库库存台账匹配规则**: 仓配商 + 仓库 + 货主 + 商品码 + 单位 + 生产日期 + 到期日 + 批次号（8项完全匹配）
